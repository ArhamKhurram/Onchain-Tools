## Trenchcord Desktop

Download the file for your operating system below and install it. Your token and data stay entirely on your own machine.

### macOS

Trenchcord isn't signed with an Apple Developer certificate yet, so macOS will block it on first launch ("Trenchcord is damaged" or "from an unidentified developer"). This is expected — clear it once:

1. Drag **Trenchcord.app** into your Applications folder.
2. Open **Terminal** (press `Command + Space`, type "Terminal").
3. Run — including the trailing space, then drag the app onto the window to fill in the path:

   ```
   xattr -cr /Applications/Trenchcord.app
   ```

4. Press Enter, then open Trenchcord. Enjoy!

The `.dmg` and `.zip` downloads both include a **READ ME FIRST.txt** with these steps.

### Windows

Windows SmartScreen may warn about an unknown publisher. Click **More info → Run anyway**.
