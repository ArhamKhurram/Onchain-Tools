OCT for macOS — First Launch
============================

OCT is not signed with an Apple Developer certificate, so macOS will
block it the first time ("OCT is damaged" or "from an unidentified
developer"). This is normal for unsigned apps.

Steps:
1. Drag OCT.app into your Applications folder (or wherever you'd like).
2. Open Terminal (Applications → Utilities → Terminal). Do not double-click
   Terminal, not the OCT app.
3. Paste this command and press Enter:

       xattr -cr /Applications/OCT.app

   (Or drag and drop OCT.app onto the Terminal window — it will fill in the path.)

4. Press Enter, then open OCT — enjoy!
